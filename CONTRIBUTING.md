# Contributing

感谢你为这个仓库做贡献。

当前仓库还处于初始化阶段，欢迎通过补充文档、完善结构、提交示例代码和提出改进建议来帮助它逐步成型。

## 适合提交的内容

- 修正文档中的错别字或表达不清的地方
- 补充项目目标、目录说明和使用说明
- 添加更明确的技术栈初始化方案
- 增加基础脚本、测试或自动化配置

## 提交前建议

请尽量确保：

- 改动目标明确
- 文档与实际仓库内容一致
- 不要提交密码、密钥、令牌或私人配置
- 不要把本地缓存、构建产物和依赖目录提交到仓库

## 推荐流程

1. Fork 仓库或新建分支
2. 完成修改
3. 自查改动内容
4. 提交 Pull Request
5. 在 PR 中说明修改目的和影响范围

## Commit 信息建议

可以参考下面的写法：

- `docs: update project description`
- `chore: improve repository setup`
- `feat: add initial project scaffold`
- `fix: correct typo in README`

## Pull Request 建议

提交 PR 时，建议说明：

- 为什么要改
- 改了什么
- 是否影响现有文件结构
- 是否需要后续继续补充

## 沟通方式

如果你不确定怎么改，建议先通过 Issue 说明想法，再开始提交代码或文档。
