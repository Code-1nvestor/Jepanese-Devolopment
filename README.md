# Jepanese-Devolopment

这是一个面向日语学习者的学习软件项目。

它的核心目标不是单纯背单词，而是帮助学习者持续提升日语口语能力，让学习从“不敢开口”逐步进步到“能听懂、能简单沟通、能复杂沟通”。

## 这个软件是干什么的

这个项目准备做成一个聚焦日语口语进步记录与训练的学习工具，重点解决下面几个真实问题：

- 学了很久日语，但不敢开口说
- 能看懂一点、背一点，但听不懂真实表达
- 只能说非常简单的句子，无法持续沟通
- 不知道自己的口语能力有没有在进步

这个软件的作用，就是帮助用户把“日语学习”变成一条看得见进步的路径。

## 核心目标

这个软件希望帮助用户逐步做到：

1. 愿意开口说日语
2. 能听懂常见、简单的日语表达
3. 能进行基础日常沟通
4. 能从简单沟通逐步走向更复杂的表达和交流

如果用一句话概括，这个项目的目标就是：

> 帮助日语学习者持续记录和推动口语能力进步，而不是只停留在“看过、背过、学过”。

## 面向谁

这个软件主要面向：

- 想提升日语口语的学习者
- 想记录自己日语进步过程的人
- 希望从“输入”走向“输出”的初学者或进阶学习者

## 计划解决的问题

很多语言学习者都有一个共同问题：

- 学习内容很多
- 真正开口很少
- 练习没有持续记录
- 不知道自己是停滞了，还是在进步

所以这个项目后续会围绕“开口说、听得懂、敢沟通、能复盘”来设计。

## 计划强调的能力方向

根据当前项目目标，后续功能会优先围绕这些方向展开：

- 口语练习记录
- 听力理解进步记录
- 从简单表达走向复杂表达的阶段性成长
- 学习过程可视化，让用户知道自己到底有没有变强

## 可以发展的功能方向

下面这些是和当前目标最贴近的功能方向：

- 每日口语练习打卡
- 口语练习内容记录
- 听力理解难度记录
- 常用表达和场景表达积累
- 阶段性进步总结
- 学习节奏追踪
